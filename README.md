# Загрузка изображений котят: синхронно vs асинхронно

Этот небольшой демонстрационный проект сравнивает два подхода к загрузке изображений из интернета:

- **Синхронный** — последовательная загрузка (одно изображение за другим)
- **Асинхронный** — параллельная загрузка с использованием `async`/`await`

Цель — наглядно показать разницу во времени выполнения при работе с сетевыми запросами.

## Используемые технологии

- Python 3.7+
- `requests` — для синхронных HTTP-запросов
- `aiohttp` — для асинхронных HTTP-запросов
